# 全面修复显示问题计划

## 📋 问题排查总结

经过全面审查，发现以下显示问题：

### 1. 信息来源标注区域（高优先级）
- **问题**: 暗色模式下背景色 `bg-gray-50 dark:bg-gray-800/50` 与文字对比度不足
- **位置**: `src/app/venues/page.tsx` 第302-320行
- **影响**: 暗色模式下文字难以辨认

### 2. CTA区域样式（高优先级）
- **问题**: 亮色模式下背景色 `bg-primary-50` 与边框色 `border-primary` 对比度不足
- **位置**: `src/app/venues/page.tsx` 第293-299行
- **影响**: 按钮区域不够醒目

### 3. 暗色模式配色优化（中优先级）
- **问题**: 部分组件在暗色模式下颜色对比度不够
- **位置**: `src/styles/globals.css` 暗色模式变量
- **影响**: 整体可读性下降

### 4. VenueHeatmap热力图（中优先级）
- **问题**: 热力图颜色在暗色模式下显示效果不佳
- **位置**: `src/components/ui/VenueHeatmap.tsx`
- **影响**: 数据可视化效果差

### 5. 训练趋势图（低优先级）
- **问题**: 柱状图在暗色模式下背景色与柱状图对比度不足
- **位置**: `src/app/profile/page.tsx` 第328-348行
- **影响**: 数据可视化效果差

### 6. 页面布局响应式（低优先级）
- **问题**: 部分页面在小屏幕下布局拥挤
- **位置**: 多个页面
- **影响**: 移动端体验不佳
